## About

We build Forge-powered apps for Jira and Confluence that help teams work smarter. Our apps are lightweight, secure, and designed to solve real workflow problems.

## Apps

| App | Product | Description |
|-----|---------|-------------|
| **JQL Calculator** | Jira | Calculate ratios, percentages, and differences between JQL query results. Display KPI metrics on your dashboard. |
| **Label Manager** | Jira | Find, rename, merge, and clean up labels across your Jira projects. |
| **Sprint Velocity Dashboard** | Jira | Compare sprint velocity across multiple boards with trend analysis. |
| **Parent-Child Status Sync** | Jira | Automatically transition parent issues when all children reach a target status. |
| **GitHub Markdown Sync** | Confluence | Sync markdown files from GitHub repositories to Confluence pages. |

## Tech Stack

- **Platform**: Atlassian Forge (serverless, hosted on Atlassian infrastructure)
- **Language**: TypeScript
- **Frontend**: Forge UI Kit 2 (`@forge/react`)
- **Runtime**: Node.js 22, ARM64
